二进制转换

  • 计算机数据的存储使用二进制补码形式存储，并且最高位是符号位
    • 正数最高位是0，负数最高位是1
  • 正数的补码与反码、原码一样，称为三码合一
  • 负数的补码与反码、原码不一样
    • 负数的原码：把十进制转为二进制，然后最高位设置为1
    • 负数的反码：在原码的基础上，最高位不变，其余为取反
    • 负数的补码：反码+1

&和&&，|和||之间的关系

  • &&表示短路且
    • 当符号左边是true时，&和&&一样
    • 当符号左边是false时，&会执行符号右边的操作，而&&不会
  • ||表示短路或
    • 当符号左边是false时，|和||一样
    • 当符号左边是true时，|会执行符号右边的操作，而||不会
// &和&&，|和||之间的关系
int i = 3;
boolean bool = true;
if (bool & (++i > 3)){
   System.out.println();
}
System.out.println(i); // 4

i = 3;
bool = false;
if (bool && (++i > 3)){
   System.out.println();
}
System.out.println(i); // 3

i = 3;
bool = true;
if (bool | (++i > 3)){
   System.out.println();
}
System.out.println(i); // 4

i = 3;
bool = true;
if (bool || (++i > 3)){
   System.out.println();
}
System.out.println(i); // 3

switch-case

  • switch中的表达式只能是特定的类型：byte，short，char，int，枚举（jdk5），String（jdk7）

  • switch中的每一对case必须对应一个break，如果忘写break则会穿透到下一个分支
int week = 5;

        break;

        break;

        break;

        break;
    case 6:
        weekStr = "Saturday";
        break;
    case 7:
        weekStr = "Sunday";
        break;
    default:
        weekStr = "error";
        break;
}
System.out.println(weekStr); // Friday

break和continue关键字

关键字 适用范围 作用
break switch-case 跳出分支
循环结构 跳出当前循环
continue 循环条件 提前进入下一次循环 
// 跳出循环
for (int i = 1; i <= 10; i++) {
    if (i > 4){
        break;
    }
    System.out.println(i);
}

System.out.println("---");

// 跳过偶数
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0){
        continue;
    }
    System.out.println(i);
}
